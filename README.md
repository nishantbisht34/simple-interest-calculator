# Simple Interest Calculator using Bash

## 📌 Project Description
This project is a simple Bash script that calculates simple interest based on user input. It demonstrates basic shell scripting and GitHub workflow practices.

## ⚙️ How It Works
The script takes three inputs from the user:
- Principal amount
- Rate of interest
- Time (in years)

It then calculates simple interest using the formula:

Simple Interest = (Principal × Rate × Time) / 100
